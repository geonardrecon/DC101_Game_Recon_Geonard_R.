# DC101_Whack_a_Mole_Recon_Geonard_R.
Whack-a-Mole is a fast-paced browser-based arcade game built with HTML, CSS, and JavaScript. Players click animated moles popping from 9 holes to score points within a 30-second timer. It tests reflexes, timing, and hand-eye coordination while teaching front-end dev skills and version control.
