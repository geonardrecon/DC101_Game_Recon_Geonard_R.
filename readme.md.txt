# Whack-a-Mole Game
A browser-based Whack-a-Mole game built with HTML, CSS, and JavaScript.

## How to Play
Click on moles as they pop up to score points. Game lasts 30 seconds!

## Features
- Timed gameplay
- Score tracking
- Responsive design
